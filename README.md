# PackageAnalyzer
Программный комплекс для анализа файлов различных типов:

1. Анализ радио сигналов
   1.1 Первичный анализ и отображение
   1.2 Удаление старт-стопных битов
   1.3 Кадровая синхронизация

2. Анализ IP пакетов.
  2.1 Анализ заголовков IPv4.
  2.2 Анализ UDP - заголовка.
  2.3 Анализ протокола NMEA0183.

3. Графическое отображение на карте NMEA - данных.
